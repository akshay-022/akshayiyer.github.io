Some things I've built

Flomi AI - Teaching AI agents

A way to instruct AI agents to do workflows using examples. People have specific knowledge, and examples seemed to be the best way to capture that.




Mirror AI - Your AI Digital Persona

A platform to create your own digital persona. It's an AI that talks like you and that you can feed your information to in an authenticated way. Allow your friends to chat with it and embed it in your own website as an API. Built for the YC + Supabase hackathon (Got a special mention).


Agent-based automation platform

Current automation platforms are almost like visual coding (blocks and flowcharts), which is too complex for the average HR/accounting person to learn and do quickly. I'm trying to build a much more intuitive automation platform for enterprises using Agentic AI and LLM Code generation.


Unit testing for LLM prompts

Created a library that captures LLM input prompts you put in during development and uses an LLM to generate variations of these input-output pairs. This enables developers to evaluate the reliability of their system prompts in different ways. Entrepreneur first hackathon project.


AI agents to automate accounts

I used AI agents to automate tedious accounting tasks like invoice data entry and ledger scrutiny. Used Langchain and LangGraph. Human in the loop for important actions.


AI assistant to learn complicated software

Many enterprise software like Epic EHR, and consumer software like Logic Pro have a very steep learning curve. I built an AI assistant that can teach you how to use it.


Innovation platform for Columbia students

I built a platform for Columbia students to share ideas to get feedback, connect with fellow builders, and collaborate on projects.




Ethereum-based freelancing platform

A decentralized freelancing platform providing secure job completion verification and automated payments for freelancers. This was FCFS as compared to other platforms connecting single employers to freelancers.
